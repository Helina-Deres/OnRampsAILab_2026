**Proof of Concept** 

**Project Overview**  
VersiGuide is an AI tinea versicolor learning assistant, developed using Microsoft Foundry, to help users learn about tinea versicolor simply and in conversation with AI, but also minimize hallucinations by using limited and trusted sources. The objective of this proof of concept is to prove that VersiGuide is an accurate and helpful learning assistant.  

**Solution Description**  
VersiGuide should define tinea versicolor, describe its usual symptoms, common locations, and general prevention strategies. It should not diagnose conditions, provide medication, nor should it respond with definitive conclusions, only possibilities. Instead, it should encourage the user to seek professional advice, like from a dermatologist. The goal is to help people learn more about tinea versicolor without worrying about being diagnosed or which sources the AI agent uses. 

**Testing**  
User   
“What is tinea versicolor?” 

AI Agent  
“Short answer  
Tinea versicolor (also called pityriasis versicolor) is a common, superficial skin infection caused by Malassezia yeasts that normally live on our skin. It makes small, slightly scaly patches that are lighter or darker than the surrounding skin, most often on the chest, upper back, neck, and upper arms.”

**Results**  
This is the shortened answer from VersiGuide’s response, but it also provided a longer and denser answer for the user, depending on the user’s preference. After receiving a prompt from the user, VersiGuide was able to simply and concisely provide an accurate response based on the sources from the National Library of Medicine. It did not diagnose the user, nor did it respond with definitive conclusions. VersiGuide simply helped the user learn more about tinea versicolor, and no other topic. 

**Conclusion**   
This proof of concept showed that VersiGuide is accurate, follows instructions clearly, and does not hallucinate. Furthermore, VersiGuide does not use other sources besides the ones I initially provided it with from the National Library of Medicine. Overall, VersiGuide is a helpful learning assistant that educates the user on tinea versicolor in a simple and reliable way.
